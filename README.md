# Online Result Processing and Grading System

## Description
The Online Result Processing and Grading System is a web-based application that allows administrators to add student details, enter marks, automatically calculate grades, and allow students to view their results using their roll number.

## Technologies Used
- HTML
- CSS
- PHP
- MySQL
- XAMPP Server

## Features
- Add Student Details
- Enter Student Marks
- Automatic Grade Calculation
- View Student Result
